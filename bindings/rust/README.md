# transcribe-cpp

Rust bindings for [transcribe.cpp](https://github.com/handy-computer/transcribe.cpp),
a C/C++ speech-to-text library built on ggml.

> **Status: placeholder.** This release reserves the `transcribe-cpp` name on
> crates.io while the first-party bindings are developed. It ships no
> functionality yet. Watch the repository for the first functional release.
