# transcribe-cpp-sys

Native FFI bindings for [transcribe.cpp](https://github.com/handy-computer/transcribe.cpp),
a C/C++ speech-to-text library built on ggml.

> **Status: placeholder.** This release reserves the `transcribe-cpp-sys` name
> on crates.io while the first-party bindings are developed. It ships no
> functionality yet. Watch the repository for the first functional release.

- Crate: `transcribe-cpp-sys` (raw FFI; the safe API will be `transcribe-cpp`)
